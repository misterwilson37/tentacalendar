New project

New projects copy the stage template (⚙️ Settings). Each project's stages are then independently editable — check any stage early; the queue only ever nags about the earliest unchecked one.

⚙️ Settings

Tiers (priority queue)

Rank 1 sorts first. Setting a tier to "Calendar" reveals a Calendar ID field — that tier then holds appointments pulled from that Google Calendar. Hover anything in a row for an explanation.

Project stage template

The universal pipeline copied into every NEW project (existing projects keep their own copy). Offset days: Before start = activates N days ahead of the project (engagement letter −14). During = activates N days after the project starts (0 = day one). After end = N days past the finish. A stage never surfaces before its predecessors are checked, whatever its date.
